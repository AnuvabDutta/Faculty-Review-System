Our project "Faculty Review System" is to show that a connection can be achieved between the faculty and students. 
This project has two main goals:
Firstly it is an easier way to select faculty during course registration based on the reviews given by the other students. 
Secondly, this system would help the faculty to change their teaching methods.

Software Requirement:

Front End:
NetBeans
Javac
Javafx
JDBC

Back End:
PHP 
MySQL

Steps:
1. Open the file "review_system" in PHP and MySql.
2.Open project folder "review_" in NetBeans. 
3. Open "admin.java" file and run it in NetBeans.
4.Then Follow as instructed in the Front End.
